## 🛠️ Quick Start

1. **Initialize the project**

```bash
pnpm run init
```

2. **Development mode**

```bash
pnpm run dev
```

Visit [localhost:8099](http://localhost:8099) to view your application

3. **Production build**

```bash
pnpm run build
pnpm run start
```
